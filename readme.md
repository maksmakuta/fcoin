# fcoin

 Microcryptocurrency with PoV consensus algorithm written in C++

## Dependencies

 - C++20
 - CMake 3.20+ (for build)
 - SQLite 3.0+
 - [zmqpp](https://github.com/zeromq/zmqpp)
 - doctest (only for tests)
