# fcoin

Cryptocurrency with Proof of Vote (PoV) consensus algorithm written in C++  
In development.  

## Dependencies

 - C++20
 - CMake 3.20+ (for build)
 - levelDB 1.20+
 - GMP 6.0.0+
 - [sockpp](https://github.com/fpagliughi/sockpp/)
 - doctest (only for tests)

 Developed and tested on ArchLinux with clang.
